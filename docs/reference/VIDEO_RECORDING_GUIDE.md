## Recording Steps

### Pre-Recording Setup

1. **Close distractions**
   - Close email, notifications, extra windows
   - Put phone on silent
   - Close unnecessary apps

2. **Test audio**
   - Record 5-second test
   - Check sound levels (not too loud, not too quiet)
   - Ensure no background noise

3. **Prepare content**
   - Have Space loaded in one browser tab
   - Have Claude Desktop open (if demoing)
   - Write down key numbers/examples
   - Have any visuals prepared

4. **Screen layout**
   - Maximize browser window
   - Use readable font size (zoom in if needed)
   - Hide sensitive info (API keys, etc.)

### During Recording

1. **Start recording**
   - Screen capture running
   - Audio input active
   - Confirm it's recording

2. **Deliver script**
   - Speak clearly and moderately
   - Go slower than normal (viewers need time to read)
   - Pause for important points
   - Don't rush through numbers/results

3. **Navigate naturally**
   - Use mouse to point at things
   - Scroll smoothly (not too fast)
   - Wait for pages to load completely
   - Show results clearly before moving on

4. **Stay on time**
   - Aim for 90-120 seconds
   - If you go over, edit in post-production
   - Leave small pauses between scenes

5. **Stop recording**
   - Include a few seconds of silence at end
   - Clearly mark where recording ends

### Post-Production (Optional but Recommended)

**If your video is too long or rough**:

Using DaVinci Resolve (free):
1. Import video
2. Cut out long pauses or mistakes
3. Add title screen at start (optional)
4. Speed up slightly if needed (1.05x-1.1x)
5. Export as MP4

## Recording Techniques

### Camera Position
- **Screen**: Centered, full-frame
- **Talking head**: Top 1/3 of frame (if including yourself)
- **Don't**: Shaky footage, too zoomed in/out

### Cursor/Mouse
- Move naturally, not too fast
- Point at important UI elements
- Highlight where users should look

### Pacing
- **Slow scene**: 3 seconds to understand
- **Fast scene**: Results, navigation
- **Important number**: Pause 2 seconds
- **Transitions**: 1 second blank

### Audio
- Clear, confident voice
- Medium pace (not rushed, not slow)
- No filler words ("um", "uh", "like")
- Background: Quiet (no music, no keyboard clicks)

## Common Mistakes to Avoid

‚ùå **Too fast** - Viewers can't read or understand  
‚ùå **Too quiet** - Audio is inaudible  
‚ùå **Distracting tabs** - Close email, other apps  
‚ùå **Long pauses** - Cut them in editing  
‚ùå **Poor lighting** - Dim screen is hard to read  
‚ùå **No script** - "Um, uh, let me think..." (use a script)  
‚ùå **Showing API keys** - Hide/blur sensitive info  
‚ùå **Too long** - Keep it under 2 minutes
